# Three-World Paper Collage Portfolio

A single responsive portfolio told through three scenic paper-collage chapters: Code, Clothing, and Life.

## Run locally

```bash
npm install
npm run dev
```

Scroll through Sunroom Meadow for computer science, High Country Postcard for clothing, and Afterglow Garden for personal interests.

## Project map

- `outputs/CREATIVE_BRIEF.md`: creative direction and concept rationale
- `AGENTS.md`: project rules and quality guardrails
- `src/data/portfolio.ts`: placeholder content to replace first
- `src/components`: shared interface and collage primitives
- `src/styles`: visual systems and responsive behavior
- `public/assets`: generated scenic artwork
